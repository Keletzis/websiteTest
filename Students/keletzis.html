Comments about my pages

As the leader of the team, I was responsible for making the home page of the website, the history and definition part and also one of the main implementations of the 4th Industrial revolution, the future of old jobs as well as the creation of new ones.

For all the html pages of the website, me as well all the rest of my collegues worked together on making a navigation bar on top of the page using html and CSS, a side menu using html CSS and Javascript and also a footer containing the most importand parts of our page. Some other things that we had in common was the up-Arrow which was made by me, and also the dark-light mode which was made by Ares.

In terms of the index home page, other than the standard html and CSS elements I created a div as a navigation bar where the user can see all the topics of the website and go to the one that he/she wishes. I had to ask my collegues for providing me a picture that would represent their topic, as well as a small sentence describing what the topic is mostly going to be focused on in order to fill it in the div. I also used some hover and transisition commands so that when you hover over the image the text appears from the bottom. You will also notice that since there are 8 topics and each row contains 3 topics, I had to make an empty div in the middle of the 3rd row in order to have the other ones better aligned.

The images are meant to represent things that are explained inside the paragraphs.
