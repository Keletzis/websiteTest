HTML

The HTML of my pages is mostly comprised of stacking divs in combination with other tags in order to be able to easily manipulate the CSS later on easier. The most “complicated” one is probably the content div since it contains multi level divs like the ones used to create images with descriptions. I also find important the fact that I am providing the user with a great number and variety of links so he can freely explore the topic he is interested in greater depth.

CSS

I created a single CSS file for all my pages(not sure if it is the best way) since they are quite similar. My CSS is responsible for almost all the styling in my site and the most important use of it for me is the fact that almost every part of my site is responsive regarding changes in the screen sizes.

JavaScript

I used JavaScript for two things. For a slide show of four images with descriptions and for an alert prompt to the user whenever he is pressing a link.
